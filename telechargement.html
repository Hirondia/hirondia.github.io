Guide d'Installation

1

Télécharger le fichier

Cliquez sur le bouton de téléchargement ci-dessus. Le fichier HirondiaPro_v2.0_Setup.exe sera téléchargé dans votre dossier Téléchargements.

2

Exécuter l'installateur

Double-cliquez sur le fichier téléchargé. Si Windows affiche un avertissement de sécurité, cliquez sur "Informations complémentaires" puis "Exécuter quand même".

3

Suivre l'assistant

Acceptez les termes de la licence, choisissez le dossier d'installation (par défaut : C:\Program Files\Hirondia Pro) et cliquez sur "Installer".

4

Lancer Hirondia Pro

Une fois l'installation terminée, lancez Hirondia Pro depuis le menu Démarrer ou le raccourci sur le bureau. La période d'essai de 30 jours démarre automatiquement.

Questions Fréquentes

Le logiciel est-il vraiment gratuit pendant 30 jours ?
Oui, totalement gratuit pendant 30 jours avec toutes les fonctionnalités débloquées. Aucune carte bancaire requise. Après l'essai, vous pouvez souscrire à une licence ou continuer en mode lecture seule.
Quels formats de fichiers sont supportés ?
Hirondia Pro supporte les fichiers Excel (.xlsx, .xls) en import et peut exporter en Excel, HTML et PDF. Tous les fichiers sont traités localement sur votre ordinateur pour garantir la confidentialité.
Le logiciel nécessite-t-il une connexion Internet ?
Non, Hirondia Pro fonctionne 100% hors ligne. Aucune connexion Internet n'est requise après l'installation. Vos données restent privées et ne sont jamais envoyées sur un serveur externe.
Comment obtenir de l'aide si je rencontre un problème ?
Notre équipe support est disponible par email (contact@hirondia.com), chat en ligne sur le site, ou via le formulaire de contact. Les utilisateurs sous licence bénéficient d'un support prioritaire sous 4h.
Les mises à jour sont-elles gratuites ?
Oui, toutes les mises à jour sont gratuites pour les utilisateurs sous licence active. Vous recevrez automatiquement les nouvelles fonctionnalités, correctifs et améliorations.
Puis-je utiliser Hirondia Pro sur plusieurs ordinateurs ?
La licence individuelle permet l'installation sur 2 ordinateurs. Pour les cabinets, nous proposons des licences multi-postes avec tarifs dégressifs. Contactez-nous pour un devis personnalisé.
